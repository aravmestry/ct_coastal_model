# Connecticut Coastal Flood Risk Model

This Streamlit application models coastal flood risk in Connecticut under
1 ft, 3 ft, and 6 ft sea level rise scenarios.

The model uses:
- Connecticut town boundary polygons
- NOAA sea level rise raster projections
- GIS spatial analysis to estimate flood exposure

Built with Python, GeoPandas, and Streamlit.
